Description

**The goal of this projects is**: 

  - DP, as Document Profile, aims to create a document that normally a RH campany always asked to the candidate fill with software like Microsoft word.

  - To study technology capable of generate PDF on client side.

  - With this project, the candidate dont need to open that kind of software (Word), only a browser, select the some fields and generate a PDF file to send to RH campany.

  - The generation of the PDF file is executed and created only on client side, which is the browser.

**Technologies**:

  - HTML5, CSS3, JavaScript

**Frameworks | Libraries | API**:

  - JSPDF, JSDF Plugin: AutoTable, jQuery, Twitter Bootstrap v3;
